## Was ist ScreenFreezer?
ScreenFreezer ist ein Multi-Monitor-Tool, das einen zweiten externen Monitor (z.B. TV oder Beamer) einfriert, während Sie auf dem Hauptmonitor weiterarbeiten können. Das Tool ist ideal für Präsentationen, Streaming oder als statisches Referenzbild. Das Tool bietet Live-Mirror-Vorschau, globale Hotkey-Steuerung, optionale Maus-Blockierung und kann wahlweise einen schwarzen Bildschirm statt des eingefrorenen Bildes anzeigen. Entwickelt für den privaten und schulischen Gebrauch.

## Anleitung
1. Lade die Datei `ScreenFreezer.exe` über den untenstehenden Link herunter.
2. (Optional) Verschiebe die Datei an einen Ort deiner Wahl (z. B. Desktop oder Programme).
3. Starte die Anwendung mit einem Doppelklick.
4. **Wichtiger Hinweis zum Start:**
   > [!IMPORTANT]
   > Da es sich um ein intern entwickeltes Tool handelt, könnte Windows beim ersten Start einen Warnhinweis anzeigen ("Der Computer wurde durch Windows geschützt"). Klicke in diesem Fall auf **"Weitere Informationen"** und dann auf **"Trotzdem ausführen"**.
