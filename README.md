## Was ist ScreenFreezer?
ScreenFreezer ist ein Multi-Monitor-Tool, das einen zweiten externen Monitor (z.B. TV oder Beamer) einfriert, während du auf dem Hauptmonitor weiterarbeiten kannst. Das Tool ist ideal für den Unterricht, Präsentationen, Streaming oder als statisches Referenzbild. Das Tool bietet Live-Mirror-Vorschau, globale Hotkey-Steuerung, optionale Maus-Blockierung und kann wahlweise einen schwarzen Bildschirm statt des eingefrorenen Bildes anzeigen. ScreenFreezer wurde für den privaten und schulischen Gebrauch entwickelt.

## Anleitung
1. Lade die Datei `ScreenFreezer.exe` über den untenstehenden [Link](https://github.com/cca-tools/screenfreezer-app/releases/latest/) herunter.
2. (Optional) Verschiebe die Datei an einen Ort deiner Wahl (z. B. Desktop oder Programme).
3. Starte die Anwendung mit einem Doppelklick.

> [!IMPORTANT]
> **HINWEIS ZUM ERSTEN START:**<br>
> Da es sich um ein intern entwickeltes Tool handelt, wird Windows beim ersten Start wahrscheinlich einen Warnhinweis anzeigen:<br>
> "_Der Computer wurde durch Windows geschützt_"<br>
> Klicke in diesem Fall auf **"Weitere Informationen"** und dann auf **"Trotzdem ausführen"**.
